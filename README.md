# Downloader

Per scaricare i files delle immagini, necessari alla corretta visualizzazione, 
lanciare in sequenza questi comandi:

```bash
# se non si ha node installato, usare n (github.com/tj/n)
# n 4.2.2
npm install
node downloader.js
```
